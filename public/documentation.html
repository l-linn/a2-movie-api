Build a server-side component of a “movies” web application.

The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

URL, Request and Response

URL Request method Response
/movies GET Return a list of ALL movies to the user
/movies/[movie name] GET Return data (description, genre, director, image URL, whether it’s featured or not) about a single movie by title to the user
/movies/genre/[genre name] GET Return data about a genre (description) by name/title (e.g., “Thriller”)
/movies/directors/[director's name] GET Return data about a director (bio, birth year, death year) by name
/users POST Allow user to log in
/users/register POST Allow new users to register
/users/[username]/profile PUT Allow users to update their user info (username)
/users/[username]/favourites POST Allow users to add a movie to their list of favorites
/users/[username]/favourites DELETE Allow users to remove a movie from their list of favorites
/users/[username]/account-setting DELETE Allow existing users to deregister
